# Numerics Codes

Techniques included:
- Bisection method,
- Newton's, Secant, False Position methods
- Fixed Point method 
## Running
Modify f(x) then just enter <br>
<code>
python *method.py* 
</code>
<br>
in terminal and replace *method* with the suitable method name. <br>
Input the boundary values or initial points as the prompt tells you.
